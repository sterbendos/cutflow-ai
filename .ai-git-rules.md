# Autonomous AI Git Rules
1. Before committing, you MUST execute compilation verification:
   - For Frontend: Run `npm run build` or `vite build`
   - For Backend: Run `cargo check` inside the src-tauri folder
2. If ANY verification check fails, do NOT commit. Fix the errors and re-test.
3. Once all checks pass cleanly, execute the following bash loop:
   `git add .`
   `git commit -m "feat(core): autonomous enhancement of codebase"`
   `git push origin main`
